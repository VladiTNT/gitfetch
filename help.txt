Gitfetch is a command-line tool that displays information about the current project directory.

By default it shows total size of the project, total amount of lines and language makeup.

Flags:
    -dir="...": specifies a subdirectory (default is ".")
    -nosize: removes the total size of the project from the info message
    -nolines: removes the total amount of lines from the info message
    -nolang: removes the langauge makeup from the info message
    -avgchars: shows the average amount of characters on each line
